Class: netlinkwrapper

netlinkwrapper

A simple synchronous TCP module wrapping the netlink library for simple IO.

new netlinkwrapper()

Constructs a netlinkwrapper, taking no arguments

Methods

boolean | undefined blocking(blocking)

Sets or gets if the socket is blocking
Name Type Description
blocking undefined | boolean optional if passed then acts as a setter boolean, if not passed then acts a getter
Returns: boolean | undefined
using as a setter, returns undefined. If using as a getter gets if the socket is blocking

connect(port, host)

A synchronous version of net.Socket.connect(port[, host]). Note: it does not accept a connectListener as part of the argument
Name Type Default Description
port number the port to connect to
host string "127.0.0.1" optional the host to connect to

disconnect()

Disconnects/Destroys the connection

string read(buffer, blocking)

Reads a socket for data. Basically a replacement for on('data');
Name Type Description
buffer number How many bytes to read from the buffer
blocking boolean optional If passed sets the blocking mode, as if you called blocking()
Returns: string
string read from the socket, or undefined if no data to read

write(data)
